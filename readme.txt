Get complete control over what categories are shown on your Blog. Easily exclude categories and display just one, or many.

== Description ==

Homepage Categories makes it easy to modify the Posts that show on your Blog/Posts page. With this plugin you can:

* remove a category
* remove many categories
* only display one category

Homepage Categories gives you a simple interface to completely control the categories you want displayed on your Blog.

== Installation ==

1. Download the plugin
1. Go to your Plugins page, then click on Add new and select the plugin's .zip file
1. Alternatively, you can extract the contents of the zip file directly to your wp-content/plugins/ folder
1. Finally, go to your Plugins page and activate Homepage Categories
